# Employee Database Management System - Intermediate Level

This project is an intermediate-level SQL-based employee database management system. It demonstrates:
- Creating tables with relationships and foreign keys.
- Inserting data into normalized tables.
- Using SQL views for simplified query access.
- Advanced SQL queries with joins, subqueries, and aggregates.
- SQL trigger for automatic logging when a new employee is added.

## Project Structure
- **tables.sql**: Contains SQL statements to create the normalized database and tables.
- **insert_data.sql**: Inserts sample data into the normalized tables.
- **queries.sql**: Includes advanced SQL queries, views, and a trigger.

## Technologies Used
- SQL (PostgreSQL)
